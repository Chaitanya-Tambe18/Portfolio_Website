#  Portfolio Website

A modern and responsive personal portfolio website to showcase my projects, technical skills, and achievements. Designed with a clean UI and smooth navigation to provide an engaging user experience.



## Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and modern user interface
- Fast loading and optimized performance
- Project showcase with live demo and GitHub links
- About Me, Skills, and Contact sections
- Deployed and accessible online


## Tech Stack

- **Frontend:** React.js, JavaScript  
- **Styling:** Tailwind CSS / CSS  
- **Deployment:** Netlify
